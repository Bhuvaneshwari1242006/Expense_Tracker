# 💰 Expense Tracker (Python)

## 📌 About
Simple Python project to track daily expenses.

## ⚙️ Features
- Add expense (category + amount)
- View all expenses
- Shows total amount
- Saves data in CSV file

## 🛠️ Tools Used
- Python
- CSV module

## 🚀 How to Run
Open terminal and type:

python expense_tracker.py

## 📂 Files
- expense_tracker.py
- expenses.csv (auto created)

## 👨‍💻 Output
User can add and view expenses easily in console.
